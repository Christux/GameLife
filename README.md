# GameLife
Python graphical Conway's Game of Life

<i>LifeGame.py</i> is the main program.</br>
<i>Cells.py</i> contains the backend methods for living cells calculation.</br>
<i>GUI.py</i> is the frontend Tkinter interface.</br>
